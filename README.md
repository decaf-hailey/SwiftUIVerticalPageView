## Requirements
- iOS 14+\n
- Swift 5+\n
- SwiftUI 2+\n 

## Installation

SwiftUIVerticalPageView is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'SwiftUIVerticalPageView'
```
